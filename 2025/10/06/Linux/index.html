🐧 我的 Linux 之旅：从“小白”到“小黑”

> 警告：以下内容可能包含大量命令行、踩坑记录和“我为什么会格式化掉整个硬盘”的血泪史。

🧭 当前进度：60%（还在爬坑中）

发行版 使用时间 当前状态 备注
openSUSE Tumbleweed 2025.03 → 至今 ✅ 备用! 滚动更新真香，滚炸也真疼
Ubuntu 22.04 LTS 2024.12 → 2025.02 ✅ 主力系统 图形界面真稳，snap 真卡
CentOS 7 2024.09 → 2024.11 ⚠️ 已退役 yum 慢到怀疑人生，但学会了 systemd

📚 一路留下的脚印

1️⃣ CentOS 7：第一次亲密接触

  • 场景：老旧笔记本装黑群晖失败，顺手装了 CentOS 7
  • 学会：
    • yum install 换阿里云源
    • systemctl start 启动服务
    • 第一次用 vi 保存文件：按了 10 分钟 ESC
  • 踩坑：
    • 把 sudo 用户删了，重启进不去系统
    • 结论：CentOS 适合服务器，不适合桌面

2️⃣ Ubuntu 22.04：图形界面真香

  • 场景：主力办公，需要微信、QQ、VSCode
  • 学会：
    • apt 三连：update | upgrade | autoremove
    • 用 gnome-tweaks 把字体换成霞鹜文楷
    • 装了 wine 跑微信，结果微信比系统还稳
  • 踩坑：
    • 点了“自动登录”，结果忘了密码
    • 结论：Ubuntu 适合新手，但 snap 是真慢

3️⃣ openSUSE Tumbleweed：滚动的快乐

  • 场景：想体验最新内核 + KDE 桌面
  • 学会：
    • zypper dup 一键滚动更新
    • snapper 回滚系统（救命神器）
    • 用 opi 装专有驱动（NVIDIA & 微信）
  • 踩坑：
    • 滚着滚着显卡驱动没了，黑屏
    • 进 snapshot 回滚，继续滚
    • 结论：Tumbleweed 适合爱折腾的人，snapper 是亲妈

场景 Ubuntu openSUSE CentOS
更新系统 apt update && apt upgrade zypper dup yum update
装软件 apt install xxx zypper in xxx yum install xxx

看日志 journalctl -xe journalctl -xe journalctl -xe

🎯 下一步计划

  • 学会用 docker + podman 做容器
  • 把 dotfiles 丢到 GitHub，一键换新机
  • 试试 NixOS，听说滚不炸？
  • 写一份「Linux 小白生存指南」放在博客
